## 📖 快速导航

### 新手上路
1. 阅读 [setup/QUICK_START.md](./setup/QUICK_START.md)
2. 按照 [setup/LOCAL_SETUP.md](./setup/LOCAL_SETUP.md) 设置本地环境
3. 配置环境变量 [setup/ENV_SETUP.md](./setup/ENV_SETUP.md)

### 数据库设置
1. 查看 [database/MIGRATION_GUIDE.md](./database/MIGRATION_GUIDE.md)
2. 运行迁移 [database/RUN_MIGRATION_GUIDE.md](./database/RUN_MIGRATION_GUIDE.md)
3. 设置 RLS [database/RLS_SETUP_GUIDE.md](./database/RLS_SETUP_GUIDE.md)

### 遇到问题？
查看 [troubleshooting/TROUBLESHOOTING.md](./troubleshooting/TROUBLESHOOTING.md) 或相关故障排除文档。
